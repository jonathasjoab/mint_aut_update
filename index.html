Atualização Automática - instalação de script

Para instalar automaticamente, execute o comando abaixo no terminal:

curl -fsSL https://jonathasjoab.github.io/mint_aut_update/update.sh | sudo bash

Ou baixe o script manualmente:

📥 Baixar setup.sh

Instruções

  • Abra o terminal: Menu Principal > Administração > Terminal
  • Cole o comando copiado (basta clicar com o botão direito do mouse, dentro do terminal e escolher a opção "Colar")
  • Digite sua senha quando solicitado
