Setup Automático Linux Mint

Para instalar automaticamente, execute o comando abaixo no terminal:

curl -fsSL https://jonathasjoab.github.io/mint_aut_update/update.sh | sudo bash

Ou baixe o script manualmente:

📥 Baixar setup.sh

Instruções

  • Abra o terminal
  • Cole o comando acima
  • Digite sua senha quando solicitado
  • Se tudo der certo, aparecerão as informações abaixo no terminal:
    • === Setup automático iniciado para usuário: teste === Created symlink /etc/systemd/system/timers.target.wants/mint-auto-update.timer → /etc/systemd/system/mint-auto-upda.timer. === Setup concluído com sucesso === Timer ativo: mint-auto-update.timer Log: /var/log/mint-auto-update.log
